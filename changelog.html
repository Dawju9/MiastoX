📜 Changelog

Historia zmian MiastoX — wszystkie wersje i aktualizacje

🔜 Unreleased

Dodano

  • Generacja bloków miejskich: budynki, rusztowania, dźwigi, mosty, schody przeciwpożarowe
  • 18+ typów obiektów miejskich (moving_scaffold, rotating_sign, collapsing_ledge, crumbling_stairs, boost_vent, bounce_tarp, rebar, elevator_shaft, wet_roof, construction_laser)
  • System pogody: 9 typów miejskich (smog, drizzle, overcast)
  • Interaktywna strona hierarchii z listą graczy (hierarchy.html)
  • Strona rekrutacji zespołu (jobs.html) z kolekcją ról
  • Strona changeloga (changelog.html)
  • API endpoint rosteru graczy (api/ranks.json)
  • Ciągła generacja miasta (5 bloków na poziom, skalowanie trudności)

Zmieniono

  • LevelGenerator.luau: pełna przebudowa z CityObjectType enum i fabrykami
  • LevelManager.server.luau: ciągła generacja bloków, usunięto specjalne poziomy
  • WeatherSystem.luau: usunięto nadnaturalne pogody (eclipse, apocalyptic, tornado, earthquake, meteor, stormy)
  • Generator.client.luau: bezpieczne guardy LightingRiver, task.spawn
  • init.server.luau: config miejski, stuby CityX systemów
  • Nav menu: dodano linki Changelog, Nasi gracze

Naprawiono

  • Symlink CityX: /Development/cityx/src → /Development/projekt/cityx/src

Planowane

  • Zamek Bogów w centrum miasta (createCastle)
  • Plac Królewski z bankomatami, leaderboardem, shopem
  • System rang administracyjnych w grze
  • Włączenie CityX systemów (PvP, broń, scoring, robbery, policja)
  • System prestiżu i achievementy
  • 4 dzielnice miasta z własnym oświetleniem

v1.0.1 — 2026-05-12

Dodano

  • Konfiguracja języka Luau (.luaurc) z aliasami modułów i trybem strict

Zmieniono

  • Aktualizacja trove do v1.3.0

v1.0.0 — 2026-04-12

Dodano

  • Integracja z CityX framework przez Wally package system
  • CI/CD pipeline z GitHub Actions (lint, build, security scan)
  • Lune build scripts dla alternatywnego systemu budowania
  • Wiele konfiguracji builda (default, dev, test)
  • Pliki konfiguracyjne: selene.toml, stylua.toml, .editorconfig, .pre-commit-config.yaml, .darklua.json
  • GitHub Pages z Jekyll (Gemfile, _config.yml)
  • Systemy gry: PauseMenu, HUD, Shop, DataStore
  • Build output: MiastoX-Final.rbxl, MiastoX-Rojo.rbxl
  • Skrypty developerskie: dev.sh, release.sh, setup.sh

Zmieniono

  • Aktualizacja inicjalizacji klienta/serwera do używania modułów CityX
  • Ulepszona struktura projektu dla modularności
  • Aktualizacja dokumentacji (index.html, about.md)

Usunięto

  • Przestarzały plik src/shared/Hello.luau
