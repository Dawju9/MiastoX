🏗️ Framework: CityX

MiastoX is built on CityX Framework - a modular game framework combining Rust and Lua technologies.

Key Framework Features:

  • Modular architecture with 50+ core modules
  • Automated CI/CD with GitHub Actions
  • Linting (Selene) & Formatting (StyLua)
  • Code preprocessing with DarkLua
  • Security scanning with TruffleHog
  • Full test coverage

📝 Development

Installation:

# Clone repository
git clone https://github.com/Dawju9/MiastoX.git
cd MiastoX

# Install dependencies
./scripts/setup.sh

# Development
npm run dev

# Build
npm run build

# Lint
npm run lint

# Format
npm run format
